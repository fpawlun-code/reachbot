Jak uruchomić?

Bot działa lokalnie na Twoim komputerze. Instalacja w 3 krokach:

# 1. Sklonuj repozytorium
git clone https://github.com/fpawlun-code/reachbot.git
cd reachbot/szczecin_scraper

# 2. Zainstaluj zależności
python -m venv venv
source venv/bin/activate # Windows: venv\Scripts\activate
pip install -r requirements.txt

# 3. Uruchom skanowanie
python main.py --industries restauracje kawiarnie --max-results 20
Wymagania: Python 3.9+, Chrome (dla Google Maps)
